## 必要条件
执行源代码需要按照以下软件。

* Python 3.x
* NumPy
* Matplotlib

※Python的版本为Python 3。

## 执行方法

前进到各章节的文件夹，执行Python命令。

```
$ cd ch01
$ python man.py

$ cd ../ch05
$ python train_nueralnet.py
```
